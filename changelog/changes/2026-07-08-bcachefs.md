- Provided a bcachefs Flatcar extension as an optional systemd-sysext image with the release. Write 'bcachefs' to `/etc/flatcar/enabled-sysext.conf` through Ignition and the sysext will be installed during provisioning. Ships both the userspace utilities (mkfs.bcachefs, mount.bcachefs, ...) and the `bcachefs.ko` kernel module built out-of-tree via the upstream DKMS harness. bcachefs support is experimental (pre-1.0 on-disk format) and bcachefs is not supported for the root partition. Requires kernel ≥ 6.16, so the extension only builds on channels whose kernel meets that floor.
